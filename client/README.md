# Kalani Garment Sell Center — EMS

Employee Management System built with MERN Stack.

## Features
- Employee CRUD
- JWT Authentication (Admin / Manager / Employee)
- Attendance Tracking
- Leave Management
- Salary & Payroll
- Sales Commission
- Reports & Analytics

## Tech Stack
- MongoDB, Express, React, Node.js

## Setup
```bash
# Server
cd server && npm install && npm run dev

# Client
cd client && npm install && npm run dev
```
